Tips untuk deploy sukses:

  • File HTML utama harus bernama index.html
  • Jika file Anda dalam format ZIP, pastikan file HTML utama berada di root folder
  • Ukuran file maksimal yang didukung adalah 10MB (batas yang disarankan untuk fungsi)
  • Netlify akan otomatis men-generate URL unik untuk website Anda
